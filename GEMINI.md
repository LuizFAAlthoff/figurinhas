# Project Instructions

## Workflows
- **Automatic Commits:** For any file modification, addition, or deletion in this repository, you MUST immediately commit the changes with a concise, descriptive commit message. Do not wait for a specific instruction to commit; it is a mandatory part of every execution cycle in this project.
- **Trade-off Analysis:** Whenever you ask a question that requires a design, technical, or business decision from the user, you MUST provide a brief analysis of the trade-offs (pros and cons) for the likely options. This helps in making informed decisions.
